# The Movies Project

사용자가 사용하는 향수의 노트들을 TOP / MIDDLE / BASE 를 기준으로 보여주고 사용자가 자신이 사용한 향수를 검색하면 사용자가 좋아했던 향수들의 노트들을 분석하여 새로운 향수를 추천해주는 서비스를 제공하는 웹 서비스를 만들고 있었다.
그러나 현재 향수들에 대한 데이터를 크롤링으로 가져오는 것에 실패하여 향수 데이터를 찾는 중이라 임시 중단 상태이다.
